Скрипты и библиотеки для разобра логов.

 - logsparseLib.py
 Библиотека с основными функциями и классами для использования в скриптах.

 - split_by_field.py
 Скрипт для разделения логов на отдельные файлы по какому-либо свойству, например по номеру сеанса.

 - analyze_background_sessions.py
 Скрипт для анализа фоновых заданий. Анализирует общее время выполнения, разделение по типам событий, контексты, и другое.
